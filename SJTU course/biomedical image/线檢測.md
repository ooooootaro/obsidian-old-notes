![[Pasted image 20240106193640.png]]
1. 水平边缘检测核： 
	[−1−2−1
	   0  0   0
	   1  2   1]
    
2. 垂直边缘检测核：
	[−1  0  1
	 −2  0  2
	 −1  0  1]⎣
    
这些核的目的是强调图像中的水平或垂直方向上的变化。

总结：拉普拉斯算子和水平、垂直边缘检测核是不同的，用途也有所不同。前者是各向同性的通用边缘检测算子，而后者是特定方向的卷积核，用于强调水平或垂直方向上的图像变化。
